Real time chat - RoR
=============================================

Real time chat running on Ruby on Rails.

This chat uses:

- Cassandra 2 (NoSQL Database)
- Cequel ORM framework for Cassandra
- Javascript, Jquery and ajax calls
- Ruby 2.1
- Rails 4
- css3
- html5
